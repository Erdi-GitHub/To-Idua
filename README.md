# To-Idua: a TODO list like Idea-manager:tm: (lol) for the command line.

## Installation
### Requirements
- g++ (installed with gcc)
- [nlohmann/json](https://github/nlohmann/json)
### Unix:
```bash
git clone https://htps://github.com/Mr-Owllers/To-Idua
cd To-Idua
make install
make clean
```
### Windows:
```
git clone https://htps://github.com/Mr-Owllers/To-Idua
cd To-Idua
install
```
